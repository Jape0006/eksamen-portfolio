Tema beskrivelse

Simpel html og css

I Tema 2 lærte jeg grundlæggende HTML og CSS, hvor vi arbejdede med simple opgaver for at skabe en forståelse for, hvordan man bygger basale websites. Vi lavede små projekter som en hjemmeside, der forklarede, hvordan man koger et æg, og en simpel to-do-liste. Her brugte vi grundlæggende HTML-elementer som p (paragraf), h1,h2 og h3 og linjeværktøjer som "li" og "ul" til at strukturere indholdet. Dette gav mig en solid introduktion til de essentielle værktøjer, der bruges til at skabe hjemmesider fra bunden.

Tema 2

Tema beskrivelse

I Tema 2 lærte vi om grundlæggende web. Vi lavede en masse øvelser der gav os basal viden om kodning og lærte os at bruge værtøjer og genveje i vs-code. Vi lavede udover det en studiestartsprøve, der gav os viden og erfaring inden for grundlæggende html og css kodning og lærte generelle css-stylings redskaber, såsom baggrundsfarve, border og knapper

Studiestartsprøve

Vi lavede en studiestartsprøve, der gav os indblik i grundlæggende html, css, grid og genveje i visual studio code. Her lærte vi også om media queries og om at kode side til både mobilsite størrelse og desktop størrelse. Vi brugte generelle css-stylings redskaber, såsom baggrundsfarve, border og knapper, dette kan også ses på blandt andet forsiden af sitet. Vi fulgte simple grid, der hjælp fra media queries skulle få sitets indhold til at passe både mobilskærme og desktop skærme.
